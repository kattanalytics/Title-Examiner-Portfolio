# California Title Research Portfolio

This repository contains a series of mock title reports created for practice and demonstration purposes.  
Each report follows professional standards for property title examination with a focus on **California oil and gas parcels**, specifically in **Kern County**.  

## Contents
- `/mock_reports` — Sample title reports prepared using public parcel data.
- `/supporting_documents` — Map screenshots, GIS references, and ALTA notes.
- `/research_notes` — Logs of parcel lookups, deeds, and other reference material.

## Tools and Data Sources
- Kern County GIS Mapping System
- California State Well Finder
- ALTA Title Standards
- Public Assessor and Recorder Data

## Purpose
To demonstrate research, documentation, and analytical skills applicable to **Remote Oil & Gas Title Examiner** roles.
